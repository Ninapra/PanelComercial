## Cómo añadir una ADR

1. Copia la última ADR y renombra con el siguiente número correlativo.
2. Título en imperativo o sustantivo descriptivo.
3. Estados: `Propuesta` → `Aceptada` / `Rechazada` / `Superada por <N>`.
4. Commit con mensaje `docs(adr): <NNNN> <título>`.
